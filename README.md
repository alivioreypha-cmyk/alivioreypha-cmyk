## 🛠️ Projects

### 🌡️ ALKESTAPI — Non-Contact Infrared Thermometer
STM32-based medical thermometer using MLX90614 sensor with TFT ST7735 display. Includes custom KiCad PCB schematic and full HAL firmware.
> `STM32` `HAL` `KiCad` `MLX90614` `C`

### 🍳 Kitch-A — Kitchen Assistant App
Flutter app with Node.js/Prisma backend on Railway + Supabase. Features barcode scanning via Open Food Facts API, shelf life tracking, and Gemini AI integration.
> `Flutter` `Node.js` `Prisma` `PostgreSQL` `Gemini API`

### 🔬 Skin Lesion Classifier
DenseNet-201 deep learning model trained on HAM10000 dataset (7 classes). Compared imbalanced vs DCGAN-balanced scenarios with 5-fold cross-validation. Includes Grad-CAM visualization.
> `Python` `Keras` `DenseNet-201` `DCGAN` `Grad-CAM`

### 🦴 FA-6500K Autorefractometer Study
Diagnostic accuracy research on autorefractometer repeatability — sensitivity, specificity, CoR analysis across ~95 subjects.
> `Medical Research` `ROC/AUC` `Diagnostic Accuracy`
